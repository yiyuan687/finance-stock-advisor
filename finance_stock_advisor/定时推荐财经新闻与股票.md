## 工作原理
1. **定时**：APScheduler 的 cron 触发器，每天 `08:00`（Asia/Shanghai）自动执行一次
2. **新闻源**：用 akshare 免费接口抓两个源（央视新闻联播 + 东方财富全球财经快讯），任一失败不影响整体
3. **推荐逻辑**：
   - 7 大板块（AI/新能源/半导体/医药/消费/金融/军工）的关键词字典
   - 扫描新闻标题+正文，命中板块关键词 → 再用利好/利空词表做情绪打分
   - 取情绪分最高且 ≥1 的板块，推荐该板块第一只龙头股
4. **输出**：控制台打印 + `output/report_YYYYMMDD.json` + `output/advisor.log`

## 安装与运行
你机器上 Python/Node 都没装，需要先装 Python（建议 3.10+）：

```powershell
# 1. 安装依赖
cd d:\chengxu

# 3. 启动每日定时（保持窗口开着，或用任务计划程序后台跑）
python finance_stock_advisor.py
```

## 关于本地 SSL 错误
本地测试时东财和央视接口都报 `SSL UNEXPECTED_EOF`，这是你本地网络波动导致的（1 小时前还能正常获取 34 条）。GitHub Actions 在云端运行，网络环境不同，通常不会有这个问题。如果 Actions 运行时也偶尔抓不到新闻，代码会日志记录并跳过，不会崩溃。
